RSA-tools - Help about random-sites

Author

2008 by Matthieu Defrance

Description

Random sites generator.

Motifs are described as position-specific scoring matrix (PSSM), where each column represents a position within the motif, and each ro a residue (A, C, G or T). The cells contain relative or absolute frequencies depending on the selected options.

The sites are generated by selecting for each column, one row at random proportionally to the column values.

Parameters

  • Sites (n): Number of sites to generate.

  • Motif (m): PSSM used to generate the sites.
