Contribution of each collection to the clusters

  • This page shows how many mofits on each cluster correspond to each collection.
  • The columns of this heatmap represent the Input Motif Databases in matrix-clustering, whereas the rows represent the clusters.
  • Each cell indicates the number motifs that a particular collection contributes to the selected cluster.

    • Dynamic Features

    1. Click on the 'Order' button to clusterize the results (by name or by a hierarchical clustering method).
    2. Click on each column label to sort its values (increasing/decreasing).
    3. Click on each row label to sort its values (increasing/decreasing).
    4. Pointing one cell will display its corresponding number of motifs.
